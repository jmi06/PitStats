About

PitRanks is an open source motorsports stats platform that calculates Driver Elo Ratings in multiple racing series. Currently, PitRanks only supports NASCAR series, but expansion to FIA series and Indycar is planned.

The Source Code is available on Github

The PitStats Elo formula is as follows:

New Rating = Old Rating + k ⋅ ( Score - Expected Score )
score = total drivers - finishing position total drivers - 1
expected_score = 1 1 + 10 Average Field Elo - Driver Elo 400
k = 16
